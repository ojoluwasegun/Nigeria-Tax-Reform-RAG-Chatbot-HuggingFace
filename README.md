# Nigeria-Tax-Reform-RAG-Chatbot-HuggingFace
This project is an intelligent Question-Answering (QA) chatbot built to help users explore and understand the Nigeria Tax Reform using Retrieval-Augmented Generation (RAG). It leverages Hugging Face models for embeddings and text generation, Flask for the backend server, and is deployable on Vercel for a fast, scalable, and serverless experience.
